Toon Boom

  • F5 : extends frame to cursor location
  • ,. : toggle between frames
  • space : pan
  • space + alt : zoom out
  • space + ctrl : zoom in
  • shift + alt + d: cuts frame and makes new drawing subsitution
  • ctrl+c ctrl+v : to duplicate drawing subsitutions
  • icon with eye that looks like a drum (red on bottom) : onion skin
  • add col-board layer for white bg (check for preset to auto add?)
  • plus / minus key : increase / decrease frame time (called exposure)
  • A : enable layer
  • D : disable layer
  • Ctrl + R : add new drawing layer

12 principles of animation

  • timing and spacing
    • time to go from pose a to pose b
    • cushioning between frames
    • ease out: object goes out (leaves first frame) slowly
    • ease in: object goes in (reaches last frame) slowly
    • ease in and ease out: starts and ends slowly
    • cut key frame timings in half

  • squash and stretch
    • stretch to give illusion of movement
    • smear with fast action but too much looks like dragging
    • squash when object "crashes" into something, more exaggerated faster object is

  • anticipation
    • action before an action
    • moving backwards before moving forward
    • anime: show anticipation action more than once in different angles

  • arcs
    • all movements follow and arc / pathway
    • avoid linear pathway, even subtle arc is okay
