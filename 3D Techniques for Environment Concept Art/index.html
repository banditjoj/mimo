3D Techniques for Environment Concept Art

  • when looking at buildings, break down into simpler parts that can be modelled and duplicated to make the whole thing, e.g. pillars, walls, decorative elements, and then how to combine with pieces from another thing e.g. from crab → the pointy elements of the claws, or texture of crab onto the texture material of the building

  • Gravity Sketch - Using Revolve tool to make different building blocks, duplicating and then adjusting the profile shape to create even more variations of the block type
  • Edge & Texture Details - from secondary ref, add detailing to building blocks
  • Import to blender - include seperate by object and group
  • Texture one piece first isolated - Textures.com - Albedo, Height + Roughness
  • (Self Notes for Texturing) - Ctrl + Shift + T import all PBR maps at once, Change Bump and Displacement in Material setting
  • Ctrl + T - generated co-ordinates & box mapping, Value node to Scale
  • F - connect two selected nodes
  • Color - to ColorRamp to adjust value range
  • Height Map to Bump Node to Normal
  • Remesh modifier (voxel, make sure no holes in mesh) → Subdiv Surface modifier (simple) and apply to add more geometry → Displace modifier

  • vertex weight proximity → make non visible object as target object → triangle icon → select all vertices and make new vertex group → modifier set Distance to Geometry and Vertex, Falloff Smooth → target object affects the weighting of the vertices, adjusting lowest and highest distance affects weight paint range, not scale of object → make top modifier on stack and can apply vertex group to displace modifier so it’s affected by weight paint

  • use ambient occlusion node + color ramp as a factor to mix different materials
  • Photoshop Texture - make half of canvas into smart object, then flip horizontal for mirroring
  • Photoshop Texture - offset 1024 x 1024 to make texture seamless → select seam then fill content aware
  • Materialize - Make different maps from Diffuse → adjust diffuse to be better for 3D usage
  • Color Ramp alternative - Gradient Texture → mix with noise map for masking textures
  • Flat Planes → Solidify + Remesh modifiers
  • Placing organic pieces with pillars and same material works very well

  • More variation - mirrored versions of kit bash, bisect mirrored versions, flip mirrored versions, etc.
  • Pieces to makes → walls, wallpieces, walkways, wall decorations, columns,
  • Have a plan before blocking out in 3D → sketch plans, gather more reference, etc.
  • A.N.T → Landscape Addon → increase subdiv → adjust mesh size to make it bigger → make multiple pieces → Billow for hills → Lakes for lakes → Mountain for mountain
  • Texturing landscape → use Google Earth Screenshot to avoid texture tiling → https://www.arcgis.com/home/webscene/viewer.html?layers=10df2279f9684e4a9f6a7f08febac2a9 → https://earthview.withgoogle.com → upscale software for bigger textures
  • Lighting → HDRI
  • Fog → Principaled Volume to World Volume → Volumetrics in Rendering to adjust fog start and end
  • Starting with basic blockout with cubes → Move them into collections → Shift + A make Collection Instance → Mirror Collection Instance so everything gets mirrored in the collection
  • Mockup Material → Right Click → Material Utilities → Select by Material
  • Copy Paste Building Blocks into scene then lay on top of simple cube blockout
